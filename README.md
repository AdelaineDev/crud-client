### Skills acquired through the project:
- Repository Pattern
- Unit Tests and Integration Tests
- Database Seeding
- Exception Handling
- GitHub Actions - Pipeline Configuration
- Semantic Commit Pattern
### Used Technologies:
JDK 17, Maven, H2 Database, Mockito, MockMVC
